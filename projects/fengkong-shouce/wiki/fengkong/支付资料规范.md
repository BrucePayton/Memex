# 支付资料规范

## 概述

支付资料规范定义了杂费付款流程中需要提交的各类凭证和文档要求，是审核环节的重要依据。

## 通用资料要求

### 基础资料
1. **付款申请单** — 包含付款事由、金额、收款方信息
2. **发票/收据** — 合法有效的付款凭证
3. **合同/协议** — 业务关系证明
4. **签报审批件** — 内部审批流程记录

### 特殊场景资料

| 场景 | 需要资料 |
|------|----------|
| 一般支付 | 付款申请单 + 发票 + 合同 + 签报 |
| 退重复缴费 | 退款申请 + 原付款凭证 |
| 投保单退款 | 退款申请 + 投保单 + 身份证明 |
| 差异退款 | 退款申请 + 差异说明 |
| 非集中退保 | 退款申请 + 退保审批件 |
| 非集中赔款 | 赔款申请 + 理赔决定 |
| 投标保证金 | 招标文件 + 保证金申请 |
| 误转款/多转款（退个人） | 退款申请 + 身份证正反面 + 签报正文一致 |
| 误转款/多转款（退公司） | 退款申请 + 公章 |

> 详细资料清单见 [[支付资料指引2025818版-湖北]]

## 资料审核标准

1. **完整性** — 所有必需资料齐全，无遗漏
2. **有效性** — 证件/合同在有效期内，签章齐全
3. **一致性** — 各资料间金额、账号、名称等信息一致
4. **合规性** — 符合财务制度和风控要求

## 与审核要点的关系

支付资料规范是 [[风控审核要点]] 的重要组成部分，审核人员需逐项核对资料是否符合本规范要求。
