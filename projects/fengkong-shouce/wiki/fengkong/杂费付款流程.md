# 杂费付款流程

## 概述

杂费付款是平安产险各机构日常运营中的核心财务流程，涉及委托方（产险机构）与被委托方（新金科财务服务中心/金服中心）之间的协作。

## 流程角色与职责

### 委托方（产险机构）
1. 发起签报申请，提供完整付款资料
2. 确保申请事项合规、金额准确
3. 对退回申请进行修正和重新提交

### 被委托方（新金科/金服中心）
1. 接收签报申请并进行初审
2. 核对付款资料完整性
3. 按审核要点逐项验证
4. 审核通过后执行付款操作
5. 对不合规申请予以退回并说明原因

## 通用审批流程

```
签报发起 → 资料提交 → 初审受理 → 审核验证 → 审批复核 → 付款执行
```
