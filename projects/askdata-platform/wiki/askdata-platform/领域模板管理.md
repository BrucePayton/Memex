# 领域模板管理

## 概述

领域模板（ChainTemplate）是 AskData 平台预置的行业分析链路模板，以 DSC YAML 格式定义分析流程。2026-05-09 完成 CRUD 能力补全，从只读展示升级为可创建、编辑、删除的完整管理系统。[^src-2026-05-09-外部资源管理重构-领域模板CRUD-开发记录]

## 数据模型

### ChainTemplate（`t_chain_template`）

| 字段 | 类型 | 说明 |
|------|------|------|
| id | Integer | 主键 |
| name | String | 模板名称 |
| description | Text | 描述 |
| dsc_flow_yaml | Text | DSC Flow YAML 内容 |
| chain_type_id | String | 链类型 ID（data/knowledge/agent/tool/business/askdata/system） |
| entry_params_schema | JSONB | 入口参数 Schema |
| domain_tags | JSONB | 领域标签列表 |
| suggest_trigger_keywords | JSONB | 建议触发关键词列表 |
| resource_requirements | JSONB | 外部资源需求声明 |
| dsc_flow_id | String | 关联的 DSC Flow ID |
| status | String | 状态：`active` / `archived` |
| creator_id | String | 创建人 ID |
| is_builtin | Boolean | 是否内置模板 |
| gmt_deleted | Timestamp | 软删除时间 |

## 后端 API

### 端点（`core/modules/chain_template/views.py`）

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/chain-template/` | 模板列表，支持 `include_builtin`、`domain_tag`、`chain_type` 过滤 |
| GET | `/chain-template/{id}` | 模板详情 |
| POST | `/chain-template/` | 创建自定义模板 |
| PUT | `/chain-template/{id}` | 更新模板（仅非内置模板） |
| DELETE | `/chain-template/{id}` | 软删除（设置 `gmt_deleted` + `status=archived`） |
| POST | `/chain-template/{id}/load` | 加载模板（返回 YAML + 参数 Schema） |
| POST | `/chain-template/{id}/push-to-dsc` | 推送至 DSC 并绑定 dsc_flow_id |
| POST | `/chain-template/suggest` | 基于问题推荐模板 |

### 业务规则

- **内置模板不可编辑/删除**：`PUT` 和 `DELETE` 端点通过 `is_builtin == False` 条件过滤
- **软删除**：DELETE 不真正删除行，而是设置 `gmt_deleted` 和 `status=archived`
- **推送 DSC**：将 `dsc_flow_yaml` 推送到 DSC `/api/flow/assets`，成功后回写 `dsc_flow_id`

## 前端实现

### DomainTemplatePage

**路径**：`web/src/features/domain-templates/pages/DomainTemplatePage.tsx`

- 内置模板与自定义模板分区展示
- 自定义模板卡片操作：使用、编辑、删除、DSC 外链
- 搜索支持：模板名称、描述、领域标签
- "创建模板" 按钮打开 `TemplateCreateDialog`

### TemplateCreateDialog

**路径**：`web/src/features/domain-templates/components/TemplateCreateDialog.tsx`

表单字段：
- 模板名称（必填）
- 链类型下拉选择（7 种：数据分析/知识检索/Agent/工具链路/业务流程/问数抽取/系统）
- 领域标签（输入后按回车添加，支持删除）
- 触发关键词（同标签交互）
- DSC Flow YAML（`<Textarea>` 编辑器，等宽字体）

### 路由

- 访问路径：`/chain-studio?tab=templates`
- `/domain-templates` 和 `/chain-workshop` 均 redirect 到 `/chain-studio?tab=templates`
