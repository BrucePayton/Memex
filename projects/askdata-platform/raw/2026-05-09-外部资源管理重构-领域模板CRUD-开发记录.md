# 外部资源管理重构 + 领域模板 CRUD + 问数空间资源扩展 — 开发记录

## 日期
2026-05-09

## 功能概述
将平台连接能力嵌入外部资源注册流程，移除独立的"平台注册"概念；每类资源（Agent/Workflow、Skill、MCP）有独立的注册交互；问数空间支持加载所有外部资源类型；领域模板支持完整的 CRUD 管理。

## 后端变更

### 数据模型
- **新增 `ExternalConnection` 模型**（`core/models/db/db_initial_models.py`）：表 `t_external_connection`，字段包含 `connection_type`（dify/coze/n8n/clawhub）、`name`、`base_url`、`auth_config`（JSONB，加密存储）、`config`（JSONB）、`is_active`
- **标记 `PlatformRegistration` 为 deprecated**，保留代码但不再使用
- **迁移文件**：`add_external_connection.py` 创建新表及索引

### API 扩展
- **外部资源 API**（`core/modules/workspace/views.py`）：
  - `PUT /external-resources/{id}` 更新
  - `DELETE /external-resources/{id}` 软删除
  - GET 增加 `category` 参数过滤（agent_workflow/skill/mcp）
- **外部连接 CRUD API**：
  - `POST /external-connections` 创建连接
  - `GET /external-connections` 列出连接
  - `PATCH /external-connections/{id}` 更新连接
  - `DELETE /external-connections/{id}` 删除连接
  - `POST /external-connections/{id}/discover` 从连接发现资源
  - `POST /external-resources/batch-import` 批量导入已发现的资源
- **问数空间**：`GET /workspaces/{id}/available-external-resources` 获取可挂载的外部资源

### 平台适配器
- **Dify 适配器增强**（`core/helpers/platforms/dify_client.py`）：
  - 新增 Console API 认证模式：POST `/console/api/login` 获取 session cookie
  - 使用 cookie 调用 `/console/api/apps` 发现应用列表
  - 向后兼容 API Key Bearer token 模式
- **`platform_service.py`**：新增 `CONNECTION_ADAPTER_MAP` 和 `discover_from_connection` 方法
- **`helper.py`**：`ResourceMountHelper` 新增 `mcp_server`/`skill`/`workflow` 挂载逻辑

### 领域模板 API
- **新增 PUT `/{template_id}`**：更新自定义模板（仅非内置模板可编辑）
- **新增 DELETE `/{template_id}`**：软删除自定义模板（设置 `gmt_deleted` + `status=archived`）
- `UpdateTemplateRequest` Pydantic 模型

## 前端变更

- **`SkillView.tsx`**：Skill Hub 连接管理、技能发现/导入、命令行安装引导
- **`MCPView.tsx`**：MCP Server CRUD、模式选择（SSE/Stdio/Streamable HTTP）
- 移除旧的 "平台管理" 按钮、扁平 KIND_LABEL tabs、JSON 配置对话框

- **移除 `PlatformManagementPage.tsx`** 及 `platforms/` 整个 feature 目录
- **移除 `ROUTES.PLATFORMS`** 路由常量

### 领域模板管理
- **`TemplateCreateDialog.tsx`**：创建/编辑模板对话框
  - 表单字段：名称、描述、链类型下拉选择、领域标签、触发关键词、DSC Flow YAML 编辑器

- **`DomainTemplatePage.tsx`**：
  - 新增 "创建模板" 按钮 + 对话框集成
  - 自定义模板卡片添加 "编辑"/"删除" 按钮（内置模板不显示）

  - CRUD 操作完成后自动刷新模板列表
- **`chain.api.ts`**：新增 `createChainTemplate`/`updateChainTemplate`/`deleteChainTemplate` API 函数

## 关键设计决策

1. **ExternalConnection 替代 PlatformRegistration**：将平台认证信息抽象为可复用的连接对象，资源通过 `connection_id` 关联连接
2. **三类分类导航**：不采用扁平的 kind tabs，而是卡片式点击进入子视图，每种资源有定制化交互
3. **非破坏性数据迁移**：保留 `PlatformRegistration` 表结构和代码（标记 deprecated），不删除旧列
4. **Dify Console API 认证**：优先使用用户名/密码登录获取 session cookie，兼容 API Key 模式
