# 数据库操作速查

> 从 Apple Notes 中提取的 PostgreSQL/MySQL 操作要点

## PostgreSQL 连接

```bash
psql -h <host> -p <port> -U <user> -d <database>
```

常用命令：
- \l 列出所有数据库
- \c <db> 切换数据库
- \dt 列出表
- \d <table> 查看表结构

## MySQL 连接

```bash
mysql -h <host> -P <port> -u <user> -p
```

## 财务数据查询模式

### 收入汇总
```sql
SELECT SUM(avenue) / 10000 AS total_income_in_wan
FROM ods_jfcxexcel_shouru_dlt_m
WHERE date_rq >= '2025-01-01' AND date_rq <= '2025-12-31'
```

### 月度趋势
```sql
SELECT DATE_FORMAT(date_rq, '%Y-%m') AS month,
       SUM(avenue) / 10000 AS total_income_in_wan
FROM ods_jfcxexcel_shouru_dlt_m
WHERE date_rq >= '2025-01-01' AND date_rq <= '2025-12-31'
GROUP BY month ORDER BY month
```

### 产品线排名
```sql
SELECT product_line, SUM(avenue) / 10000 AS total_income_in_wan
FROM ods_jfcxexcel_shouru_dlt_m
GROUP BY product_line ORDER BY total_income_in_wan DESC
```

### 同比增长
```sql
SELECT product_line,
       (SUM(CASE WHEN YEAR(date_rq) = 2025 THEN avenue ELSE 0 END) -
        SUM(CASE WHEN YEAR(date_rq) = 2024 THEN avenue ELSE 0 END))
       / NULLIF(SUM(CASE WHEN YEAR(date_rq) = 2024 THEN avenue ELSE 0 END), 0) * 100
       AS growth_rate
FROM ods_jfcxexcel_shouru_dlt_m
GROUP BY product_line
```

## 数据源表

- ods_jfcxexcel_shouru_dlt_m — 财服收入明细表
- 关键字段: date_rq, product_line, avenue, area, pay, type1

## 注意事项

- 金额字段需 /10000 转换为万元
- 除法运算注意 NULLIF 防除零
