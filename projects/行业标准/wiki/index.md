# 行业标准知识库

欢迎使用行业标准知识库，本项目专注于行业划分及相关标准规范的管理。

## 项目说明

本知识库包含行业分类标准、划分规则、内容规范及相关应用案例。

## 原始资料 (raw)

当前已上传的原始资料：
- **2017年国民经济行业划分.pdf** - 国家统计局发布的国民经济行业分类标准

## 目录结构

```
行业标准/
├── wiki/          # 知识库页面（整理后的知识内容）
├── raw/           # 原始参考资料（PDF、文档等）
└── project.json   # 项目配置文件
```

## 使用说明

1. `raw/` 目录用于存放原始文件，保持文件原貌不变
2. `wiki/` 目录用于存放整理后的结构化知识内容
3. 所有引用原始资料的内容需要标注来源
