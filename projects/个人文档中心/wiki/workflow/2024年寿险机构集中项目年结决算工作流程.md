# 2024年寿险机构集中项目年结决算工作流程

[[source-2024-life-insurance-year-end|2024年寿险机构集中项目年结决算工作流程]] 版本 v1.0，由金服财务服务中心管理，金服产品线负责，定义了2024年寿险机构集中项目年结决算工作流程，包含预算、费用、总账、税务等模块的链式任务推进。[^src-2024-life-insurance-year-end]

## 流程概述

该流程按时间轴顺序链式推进，关键节点以截止时间为准：

1. **启动年结工作**：启动2024年寿险机构集中项目年结。[^src-2024-life-insurance-year-end]
2. **预算 - E采费用类下单截止**：E采购系统下单，确保12月19日18:00前完成，责任主体为各机构预算负责人。[^src-2024-life-insurance-year-end]
3. **费用 - 后援AP扫描截止**：后援AP系统扫描，确保12月25日18:00前完成，责任主体为各机构费用经办人。[^src-2024-life-insurance-year-end]
4. **费用 - 资产凭证扫描截止**：各机构资产凭证扫描，确保12月28日前完成，12月28日起中心不再接受扫描。[^src-2024-life-insurance-year-end]
5. **费用 - NEWFAS报销模块关闭及损益计提**：NEWFAS报销模块关闭，确保12月31日18:00前完成，同时完成损益计提。OF系统接收NEWFAS导出的GL数据用于过账，年结期间数据同步频率调整为每5分钟一次。[^src-2024-life-insurance-year-end]
6. **预提类事项判断**：根据《预提账务处理办法(2024版)》执行预提，差旅费、市内交通、招待费原则上不予预提，如确需预提需总部同意邮件或签报。需补充处理则由远程费用端与预提负责人协同完成。[^src-2024-life-insurance-year-end]
7. **总账 - LAS/CDS/TMS系统期间关闭**：1月1日完成LAS、CDS、TMS系统期间关闭，需检查未制证单据。2日起未经总公司同意不得对12月数据进行操作。[^src-2024-life-insurance-year-end]
8. **TMS系统回销与制证专项流程**：TMS系统特殊要求——12月31日前完成费用付款批次回销，1月2日前完成制证上传至OF，检查已回盘但未资金确认的数据。[^src-2024-life-insurance-year-end]
9. **TMS系统最终检查**：确认无未处理异常数据。[^src-2024-life-insurance-year-end]
10. **总账 - 往来确认、税金计提与总账期间关闭**：1月3日完成往来自动制证确认、增值税附加计提、总账期间关闭。涉及财智云费用分摊模块、OF系统、LAS/CDS/TMS系统、综合财务平台、单证系统。1月4日起不得在12月期间创建任何往来流程。[^src-2024-life-insurance-year-end]
11. **年结工作结束**：流程终止。[^src-2024-life-insurance-year-end]

## 关键时间节点

| 节点 | 截止时间 | 模块 |
|------|----------|------|
| E采购费用类下单 | 12月19日18:00 | 预算 |
| 后援AP扫描 | 12月25日18:00 | 费用 |
| 资产凭证扫描 | 12月28日 | 费用 |
| NEWFAS报销关闭 | 12月31日18:00 | 费用 |
| TMS费用付款回销 | 12月31日 | 总账 |
| LAS/CDS/TMS系统关闭 | 1月1日 | 总账 |
| TMS制证上传至OF | 1月2日 | 总账 |
| 往来确认、税金计提、总账关闭 | 1月3日 | 总账 |

## 涉及系统

- **E采购系统**：预算模块下单
- **后援AP系统**：费用扫描
- **NEWFAS报销系统**：报销模块关闭与损益计提
- **OF系统**：GL数据过账
- **LAS系统**：总账期间关闭
- **CDS系统**：总账期间关闭
- **TMS系统**：总账模块与回销专项流程
- **财智云费用分摊模块**：内部往来流程处理
- **综合财务平台**：凭证检查
- **单证系统**：凭证检查

## 风险点

- **E采购下单延迟**（高风险）：影响后续费用、总账模块，需系统自动24小时预警。[^src-2024-life-insurance-year-end]
- **NEWFAS模块关闭前数据处理失败**（高风险）：影响损益数据准确性，需关闭前强制检查。[^src-2024-life-insurance-year-end]
- **系统关闭前未检查未制证单据**（高风险）：引发总账不平与审计风险。[^src-2024-life-insurance-year-end]
- **TMS制证未在截止日前上传**（高风险）：影响总账闭合，需设置上传成功通知机制。[^src-2024-life-insurance-year-end]
