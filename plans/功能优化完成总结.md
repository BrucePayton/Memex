## 完成的 Quick Wins

### 1. Raw 文件上传 ✅

**功能描述**

- 支持多文件同时选择上传

### 2. Git Revert UI ✅

**功能描述**
- History 标签页显示 Ingest 历史表格
- 每行有「Revert」按钮
- 点击后显示确认对话框
- 成功后自动刷新页面和侧边栏

**实现细节**
- 后端: `/api/revert` (已存在)
- 前端: `showHistory()`, `doRevert()` (已完整实现)

### 3. 全文搜索 ✅

**功能描述**
- Search 标签页提供 TF-IDF 全文搜索
- 显示匹配页面标题、文件名、得分和文本片段
- 点击结果直接跳转到对应页面

**实现细节**
- 后端: `do_search()` 使用 `_tfidf_wiki_search()` (已完整实现)
- 前端: `showSearch()`, `doSearch()` (已完整实现)

## 使用方式

### 新 Raw 上传功能

1. 访问 http://localhost:8090
2. 切换到任意项目
3. 点击左侧「Raw」标签
4. 点击顶部「Upload Files」按钮
5. 选择文件（支持多选），上传完成后自动刷新

### History Revert

1. 点击左侧菜单 Browse → History
2. 点击任意 Ingest 记录的「Revert」按钮
3. 确认后完成回滚

### 全文搜索

1. 点击左侧菜单 Browse → Search
2. 输入关键词
3. 点击搜索按钮或回车
4. 点击结果跳转到页面
